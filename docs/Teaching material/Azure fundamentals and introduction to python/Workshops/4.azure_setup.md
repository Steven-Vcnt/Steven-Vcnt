# Azure Setup

## Set up your Azure Student Account

Azure Student Account enables you to use free services on Azure and start to build in the cloud. You will have $100 Azure credits for the academic year.

To create a new Azure account, you will need your BSB mail address and follow the link:
[Azure for Students â€“ Free Account Credit.](https://azure.microsoft.com/en-us/free/students/)

## Set up a monthly budget

Budget in Azure aims to limit the spending of your Azure credits as you can easily spend $100 with a bad configuration.
To avoid this, we need to configure a budget, you can follow the [tutorial to create a budget.](https://learn.microsoft.com/en-us/azure/cost-management-billing/costs/tutorial-acm-create-budgets)

## Set up a SQL Database with Azure

### Azure SQL configuration

In the [portal Azure](https://portal.azure.com/#home), search for ![Azure SQL](../../../assets/images/Azure_sql/azure_sql_logo.png){ width="600"}

Click on Azure SQL Database, try it for free:
![Create azure db](../../../assets/images/Azure_sql/create_sqsl_database.png){ width="600" }

Then fill-in the configuration, select your Subscription, an existing Resource Group and create a Database name.
![Create azure db config](../../../assets/images/Azure_sql/create_sql_db_config.png){ width="600" }

Create a new Server with a Server name and choose a West Europe Location. Select an Authentication method, use Microsft Entra-only authentication.
![Create azure db server config](../../../assets/images/Azure_sql/create_sql_server_db.png){ width="600" }

In the Cost summary, verify that you have a Free estimated total.

Click on Review + create.
